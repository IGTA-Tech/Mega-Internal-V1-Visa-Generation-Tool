## 2. Tier Classification Reference

### 2.1 Tier Definitions

| Tier | Description | Examples | Weight |
|------|-------------|----------|--------|
| **Tier 1** | Major global/national media with established editorial standards | ESPN, BBC, CNN, NYT, AP, Reuters, Forbes, WSJ | Gold |
| **Tier 2** | Industry-specific authoritative sources | Billboard, SportsPro, Athletic, trade publications, major federation sites | Strong |
| **Tier 3** | Supplementary/regional sources | Local news, blogs, specialty sites, regional publications | Supporting |
| **Tier 4** | Unreliable/self-promotional | Personal websites, unverified blogs, press releases | Exclude |

### 2.2 Field-Specific Tier 1 Sources

**Athletics:**
- ESPN, BBC Sport, Sports Illustrated, The Athletic
- Official league sites (NBA.com, UFC.com, FIFA.com)
- Wire services (AP, Reuters, AFP)

**Tech/Business:**
- Wall Street Journal, Forbes, Bloomberg, TechCrunch
- Wired, MIT Technology Review, Harvard Business Review

**Sciences:**
- Nature, Science, Nature News, Science News
- New York Times Science, Scientific American

**Arts/Entertainment:**
- Billboard, Variety, Hollywood Reporter
- New York Times Arts, Rolling Stone, Pitchfork

## 5. Tier Deficit Recovery Protocols

### 5.1 When Below Tier 1 Target (26)

**Immediate Actions:**

1. **Targeted Tier 1 Search**
   - Search ESPN + beneficiary name
   - Search BBC + beneficiary name
   - Search CNN + beneficiary name
   - Search NYT + beneficiary name
   - Search AP News + beneficiary name

2. **Wikipedia External Link Mining**
   - Find beneficiary's Wikipedia page
   - Check all external links/references
   - Follow sources to find Tier 1 coverage

3. **Archive.org Historical Search**
   - Search Wayback Machine for beneficiary name
   - Check archived versions of major sites
   - Recover deleted coverage

4. **Wire Service Search**
   - Search AP Archive
   - Search Reuters Archive
   - Check AFP (if applicable)

5. **Major Event Coverage**
   - Find coverage of events beneficiary participated in
   - Check if beneficiary mentioned in Tier 1 event coverage

### 5.2 When Below Tier 1-2 Ratio (65%)

**Options:**

1. **Increase Tier 1-2 sources** (preferred)
   - Use targeted searches above
   - Check industry publications
   - Search trade media

2. **Decrease Tier 3 reliance**
   - Remove weakest Tier 3 sources
   - Replace with Tier 1-2 equivalents
   - Keep only essential Tier 3

3. **Document mitigation** (if cannot improve)
   - Note in evidence gap analysis
   - Prepare expert letter strategy
   - Brief explains source context

### 5.3 When Total URLs Below 40

**Recovery Protocol:**

1. **Expand search queries**
   - Try alternative name spellings
   - Search nicknames/aliases
   - Search in multiple languages

2. **Expand source types**
   - Add official federation/organization sources
   - Include verified social media posts (as supporting)
   - Add conference/event documentation

3. **Check related coverage**
   - Find coverage of events/teams/organizations
   - Look for beneficiary mentions in broader coverage
   - Check co-participants' coverage for mentions

4. **Expert letter supplementation**
   - More expert letters to fill gaps
   - Experts can attest to facts with limited documentation

## 7. URL Verification Checklist

### 7.1 For Every URL

Before including in final count:

- [ ] URL is accessible (not 404)
- [ ] Content matches description
- [ ] Tier classification is correct
- [ ] Source is independent
- [ ] Publication date verified
- [ ] Archive.org backup exists
- [ ] No duplicate of another URL
- [ ] Correctly assigned to criterion

### 7.2 Verification Failures

If URL fails verification:

1. **Remove from count**
2. **Find replacement**
3. **Update tracking sheet**
4. **Re-calculate ratios**

---

## 8. Final Quality Gate

### 8.1 Pre-Document Generation Checklist

Before proceeding to document generation:

**Quantity:**
- [ ] Total URLs ≥ 50
- [ ] Phase 1 contribution ≥ 15
- [ ] Phase 2 contribution ≥ 20
- [ ] Phase 3 contribution ≥ 15

**Quality:**
- [ ] Tier 1 count ≥ 26
- [ ] Tier 2 count ≥ 18
- [ ] Tier 3 count ≤ 8
- [ ] Tier 4 count = 0
- [ ] Tier 1-2 ratio ≥ 65%

**Coverage:**
- [ ] 3+ criteria with strong evidence
- [ ] Each claimed criterion has 3+ URLs
- [ ] No criterion relies on single source
- [ ] Evidence spans multiple years

**Verification:**
- [ ] All URLs verified accessible
- [ ] All claims cross-referenced
- [ ] No contradictions identified
- [ ] Independence confirmed

### 8.2 Go/No-Go Decision

| Status | Action |
|--------|--------|
| All checks pass | Proceed to document generation |
| 1-2 minor fails | Proceed with caution, note gaps |
| 3+ fails or critical fail | Return to research phase |
